# Bloker63
Bloker, or Baloker, is a combination of Texas Hold 'em Poker 
with the Joker cards from Balatro.

Under `Bloker63/` you will find the python section,
containing all the requirements for an in-person game
of Baloker.
This includes the ability to draw random jokers
and yeah that's about it. It has the original Balatro sounds.
You can run it with `python -m Bloker63`.

Under `Baloker Online/` is the real Baloker game with full
multiplayer online gameplay and stuff or whatever.
Actually at the time of writing this I haven't even started
working on it so who knows what'll be in there.
As with all great things, Baloker Online will be written in C,
so what will most likely be in there are lots of memory leaks.
